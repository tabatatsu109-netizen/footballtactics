# Tactical Coach Board ⬡

**サッカーコーチ専用 戦術シミュレーター**

フォーメーションの可視化、戦術パターンのアニメーション再生、パターン保存が可能なPWAアプリです。

## 機能

- **5種類のフォーメーション**: 4-4-2 / 4-2-3-1 / 4-3-3 / 3-4-2-1 / 3-5-2
- **攻撃5パターン**: ビルドアップ / サイド前進 / 中央突破 / サイドチェンジ / 背後への抜け出し
- **守備5パターン**: 前線プレス / ミドルブロック / ローブロック / ボールサイド圧縮 / 逆サイドを捨てる
- **アニメーション再生** (速度調整可能)
- **ドラッグ操作** (選手・ボール移動)
- **パターン保存** (localStorage)
- **PWA対応** (オフライン使用可能)
- **スマホ・タブレット対応**

## GitHub Pages での公開

1. このフォルダを GitHub リポジトリにアップロード
2. Settings → Pages → Source: Deploy from a branch → main / root
3. `https://<username>.github.io/<repo>/` でアクセス

## ファイル構成

```
├── index.html      # メインHTML
├── style.css       # スタイル
├── app.js          # アプリロジック
├── manifest.json   # PWAマニフェスト
├── sw.js           # Service Worker
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

## 操作方法

1. 自チーム・相手チームのフォーメーションを選択
2. フェーズ（攻撃/守備）を選択
3. 戦術パターンを選択
4. ▶ 再生 ボタンで選手がアニメーションで動く
5. 選手・ボールはドラッグで自由移動
6. 💾 ボタンでパターン保存
